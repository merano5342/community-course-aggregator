狀態標示，比 Tag 更小巧精緻。用於課程狀態、系統訊息、可報名/已截止提示。

```jsx
<Badge variant="accent" dot>進行中</Badge>
<Badge variant="success">可報名</Badge>
<Badge variant="warning">名額有限</Badge>
<Badge variant="error">已截止</Badge>
<Badge variant="neutral">預告</Badge>
<Badge variant="dark">精選</Badge>
```

**variant**: `accent` / `dark` / `neutral` / `success` / `warning` / `error`
**dot**: 顯示 6px 圓點狀態指示
