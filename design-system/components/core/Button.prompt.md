主要互動元素。用於表單送出、頁面 CTA、行內操作按鈕。

```jsx
<Button variant="primary">報名課程</Button>
<Button variant="dark" size="lg">立即瀏覽</Button>
<Button variant="ghost" pill>更多篩選</Button>
<Button variant="accent" size="sm">儲存收藏</Button>
<Button disabled>已額滿</Button>
```

**Variants**: `primary`（藍綠填色）/ `dark`（近黑填色）/ `ghost`（透明外框）/ `accent`（淺藍綠底）/ `surface`（暖灰底）
**Sizes**: `sm`（32px）/ `md`（40px）/ `lg`（48px）
**pill**: 膠囊圓角，適合篩選標籤型按鈕
