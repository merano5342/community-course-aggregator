# 課程列表頁

---

**頁面位置**：課程瀏覽  
**頁面路由**：`/courses`（v1.1；v1.0 為 `/`）  
**所屬功能模組**：2.0 前端核心  
**最後更新**：2026-06-20（路由遷移、行動版篩選 Drawer）

---

## 需求背景

課程列表頁為使用者瀏覽、搜尋、篩選多所社大課程的主要介面，並快速決定是否收藏或加入比較清單。資料來源為 GitHub 靜態 JSON，透過 TanStack Query 載入並快取；所有篩選與排序在 client-side 完成，無需額外 API。

---

## 路由異動說明

v1.0 中，課程列表為根路由 `/`。v1.1 起根路由改為首頁公告頁（[0.2-首頁公告頁.md](../0.0-全局功能/0.2-首頁公告頁.md)），課程列表移至 `/courses`。站內所有「前往課程列表」的 `Link` 皆使用 `to="/courses"`。

## 搜尋框

| 元素 | 說明 |
|------|------|
| **輸入框** | placeholder：「搜尋課程名稱或老師…」 |
| **即時過濾** | 對 `name` + `teacher[]` 做 case-insensitive 比對 |
| **清除按鈕** | 輸入有值時顯示 ✕，點擊清空 |
| **結果統計** | 搜尋框下方顯示「共找到 {N} 門課」 |
| **行動版篩選按鈕** | 搜尋框右側，顯示已套用篩選數量；點擊開啟篩選 Drawer |

---

## 篩選側邊欄（桌面）/ 篩選 Drawer（行動）

### 篩選區塊

| 篩選項目 | 元件類型 | 說明 |
|---------|---------|------|
| **排序** | Select | 開課日期 / 費用低→高 / 費用高→低 / 學校 |
| **學校** | Checkbox 多選 | 列出已載入學校，附彩色圓點 |
| **星期** | Toggle Chip 多選 | 日～六 7 個 |
| **時段** | Toggle Chip 多選 | 上午 / 下午 / 晚上 |
| **開課狀態** | Checkbox 多選 | 開放報名 / 名額已滿 / 取消 / 延後 |
| **課程類別** | Checkbox 多選 | 依資料動態產生 |
| **特殊標記** | Checkbox | 僅顯示新課程 / 僅顯示混成課程 |

### 篩選邏輯

- 同一篩選項目內為 **OR**（南港 OR 文山）
- 不同篩選項目之間為 **AND**（南港課程 AND 星期三）
- 所有篩選在 `useMemo` 內計算，任一條件改變立即重算

### 行動版篩選 Drawer

- 點擊搜尋框旁「篩選」按鈕開啟
- 從左側滑入（`position: fixed, left: 0, top: 0, bottom: 0`），寬度 85%，最大 320px
- 半透明遮罩（點擊關閉）
- 底部「套用篩選（N 門）」按鈕（N = 目前符合筆數）
- `FilterSidebar` 以 `embedded` prop 嵌入（去除固定寬度，改為 100%）

## 課程卡片（CourseCard）

### 卡片元素

| 元素 | 說明 |
|------|------|
| **課程名稱** | 最多 2 行 |
| **學校徽章** | 彩色 Badge，各校固定顏色 |
| **老師名稱** | 多人以「、」連接 |
| **星期 + 時段** | 「每週三 晚上」格式 |
| **費用** | NT$ {fee}，有折扣顯示劃線原價 |
| **開課日期** | `{startDate}（共 {weeks} 週）` |
| **狀態 Badge** | 開放報名 / 名額已滿 / 已取消 / 延後 |
| **新課程 / 混成 Badge** | 條件顯示 |
| **加入最愛** | 愛心圖示，已收藏時填滿 |
| **加入比較** | 比較圖示，清單滿時 disabled |

### 卡片互動

| 互動 | 行為 |
|------|------|
| 點擊卡片本體 | 開啟課程詳細 Modal（2.2） |
| 點擊最愛按鈕 | 切換收藏，Toast 通知 |
| 點擊比較按鈕 | 切換比較清單，Toast 通知 |

## 空狀態

| 情境 | 說明 |
|------|------|
| **資料載入中** | 文字「載入中…」（未來可改骨架屏） |
| **篩選無結果** | 插圖 + 「找不到符合條件的課程」+ 清除篩選按鈕 |
| **資料為空** | 「目前無課程資料，請等待每週更新」 |

## 資料載入

- 各校 JSON 路徑：`https://cdn.jsdelivr.net/gh/{repo}/data/courses/{school}.json`
- TanStack Query，`staleTime: 1 小時`
- 初期使用 `src/data/mockCourses.ts`（18 筆含完整 schema 的模擬資料）

---

## 操作流程

### 主要流程

1. 使用者點擊「瀏覽課程」或導覽至 `/courses`
2. 載入課程資料，顯示課程卡片 Grid
3. 使用搜尋框或篩選縮減結果
4. 點擊卡片開啟課程詳細 Modal（→ 2.2）
5. 在 Modal 或卡片上點擊愛心加入最愛
6. Toast 顯示「已加入最愛」

### 加入比較流程

1. 點擊卡片比較按鈕，CompareBar 從底部（行動版在底部導覽列上方）出現
2. 最多選 3 門，第 4 門 Toast 警告
3. 點擊 CompareBar「前往比較」導至 `/compare`
