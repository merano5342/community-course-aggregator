# 課程詳細 Modal

---

**頁面位置**：覆蓋層（Overlay），從課程列表頁或我的最愛頁觸發  
**頁面路由**：無獨立路由  
**所屬功能模組**：2.0 前端核心  
**最後更新**：2026-06-20（v1.1：新增 Tabs、豐富課程欄位、行動版 Bottom Sheet）

---

## 需求背景

課程詳細 Modal 讓使用者不離開列表頁即可查看完整課程資訊，包含課程簡介、逐週大綱、講師介紹、名額進度、報名連結等，對應 frog.tw 課務系統的課程詳細頁所呈現的資訊深度。採用 Overlay 模式以保留篩選狀態，行動版改為底部浮出的 Bottom Sheet。

---

## 觸發方式

| 觸發位置 | 觸發動作 |
|---------|---------|
| 課程列表頁卡片 | 點擊卡片本體（非最愛/比較按鈕） |
| 我的最愛頁列表項 | 點擊列表項本體 |
| 時間表頁事件 | 點擊週曆上的課程事件 |

```
┌────────────────────────────────────────────┐
│             ── 拖曳把手 ──                  │
│                                            │
│  [學校] [狀態] [混成] [NEW]    ×           │
│  課程名稱                                  │
│  類別                                      │
│                                            │
│  [課程資訊]  [課程大綱]                    │
│  ─────────────────────────────────────     │
│  （Tab 內容）                              │
│                                            │
│  [加入最愛]  [加入比較]  [前往報名]        │
└────────────────────────────────────────────┘
```

行動版（< 768px）：`position: fixed`，`left/right/bottom: 0`，`top: auto`，`borderRadius` 僅上方兩角，`maxHeight: 92vh`，可滾動，頂部有拖曳把手視覺。底部內容區加 `padding-bottom: max(var(--space-3), env(safe-area-inset-bottom))`。

## Tab 結構

| Tab | 說明 | 顯示條件 |
|-----|------|---------|
| 課程資訊 | 基本資訊 + 名額進度條 + 備註 | 永遠顯示 |
| 課程大綱 | 課程描述、目標學員、逐週大綱、講師介紹 | 只有當課程具有豐富內容（`description` 或 `outline` 或 `teacherBio` 任一存在）時顯示 Tab 列；否則僅顯示課程資訊 |

預設開啟：課程資訊。

---

## Tab 1：課程資訊

### 名額進度條

僅在 `quota` 存在時顯示：

```
名額  ████████░░  32 / 40 人
```

| 元素 | 說明 |
|------|------|
| **進度條** | `enrolled / quota * 100%`，以 `var(--color-accent)` 填色 |
| **人數文字** | 「{enrolled} / {quota} 人」，填滿時文字改為 accent 色 |
| **填滿時** | 進度條滿色 + 「名額已滿」提示 |

### 資訊 Grid（2 欄，行動版 1 欄）

| 欄位 | 資料來源 | 顯示格式 |
|------|---------|---------|
| 授課老師 | `teacher[]` | 多人以「、」連接；空顯示「－」 |
| 上課星期 | `dayOfWeek` | 「每週{星期}」 |
| 費用 | `fee` + `discount` | NT$ {fee}；有折扣顯示優惠標籤 |
| 上課時段 | `timeSlot` | 上午 / 下午 / 晚上 |
| 開課日期 | `startDate` | YYYY/MM/DD |
| 總週數 | `weeks` | 共 {weeks} 週（無此欄不顯示） |
| 上課地點 | `location` | 完整地點 |
| 課程類別 | `category` | 層級以 `>` 分隔 |

### 備註

`notes` 存在時顯示備註區塊（`whiteSpace: pre-line`，保留換行）。

---

## Tab 2：課程大綱

### 課程描述

`description` 欄位存在時顯示段落文字。

### 目標學員

`targetAudience` 存在時顯示：

```
目標學員
─────────────────
適合對 ○○ 有興趣的初學者…
```

### 逐週大綱

`outline[]` 存在時顯示有序清單：

```
課程大綱
─────────────────
● 第 1–2 週｜課題名稱
● 第 3–4 週｜課題名稱
…
```

每項前置 accent 圓形標記（`var(--color-accent)`），標題粗體。

### 講師介紹

`teacherBio` 存在時顯示講師卡片：

```
┌────────────────────────────────┐
│  [姓名首字母 Avatar]  授課老師名稱 │
│  講師簡介文字（whiteSpace:pre-line）│
└────────────────────────────────┘
```

Avatar 為圓形，背景 `var(--color-accent)`，顯示老師名稱第一個字。

## 頂部標識區（共用）

| 元素 | 說明 |
|------|------|
| **學校徽章** | 彩色 Badge，各校固定顏色，顯示學校短名 |
| **狀態 Badge** | 開放報名（綠）/ 名額已滿（紅）/ 已取消（灰）/ 延後（黃） |
| **新課程 Badge** | `isNew === true` 時顯示「NEW」 |
| **混成 Badge** | `isMixed === true` 時顯示「線上+實體」 |
| **關閉按鈕** | 右上角 ✕，或點擊 backdrop（桌面）關閉 |

---

## 操作按鈕區

| 按鈕 | 樣式 | 行為 |
|------|------|------|
| **加入最愛** | Secondary，已收藏時填色 | 切換收藏，Toast 通知 |
| **加入比較** | Secondary，已選時填色，清單滿時 disabled | 切換比較，Toast 通知 |
| **前往報名** | Primary + 外部連結圖示 | 新分頁開啟 `detailUrl`（原校 frog.tw） |

- `status === 'cancelled'`：「前往報名」disabled，tooltip「課程已取消」
- `detailUrl` 為空：「前往報名」disabled，tooltip「暫無報名連結」

---

## 關閉方式

| 方式 | 行為 |
|------|------|
| 點擊 ✕ 按鈕 | 關閉 Modal |
| 點擊 Backdrop（桌面） | 關閉 Modal |
| 按 Escape 鍵 | 關閉 Modal |

---

## 業務邏輯

- 使用 `createPortal` 將 Modal 掛載至 `document.body`，脫離正常 DOM 流
- 同一時間只能開啟一個 Modal
- 開啟時 `body overflow: hidden` 防止背景捲動
- 行動版不顯示 backdrop 點擊關閉（避免誤觸）

## 響應式設計

| 裝置 | 樣式 |
|------|------|
| 行動（< 768px） | Bottom Sheet：`position:fixed, left/right/bottom:0, top:auto`，`maxHeight:92vh`，上方兩角 `borderRadius`，拖曳把手，內容可滾動 |
| 桌面（≥ 768px） | 置中 Dialog：`maxWidth:600px`，半透明 backdrop，`borderRadius: var(--radius-xl)` |

## 課程豐富欄位（v1.1 新增）

對應 `src/types/course.ts`（Course interface）：

| 欄位名稱 | TypeScript 類型 | 來源（frog.tw） | 說明 |
|---------|---------------|----------------|------|
| `description` | `string?` | 課程簡介 > 理念 + 目標 | 課程描述文字 |
| `outline` | `string[]?` | 課程大綱表格（週次 + 主題） | 逐週大綱，如「第 1–2 週｜五十音入門」 |
| `targetAudience` | `string?` | 課前資訊 > 適合對象 | 目標學員描述 |
| `teacherBio` | `string?` | 講師介紹 > 簡介 | 講師文字簡介 |
| `notes` | `string?` | 課前資訊 > 其他 | 備註、攜帶物品等 |
| `quota` | `number?` | 招生人數 | 最大名額 |
| `enrolled` | `number?` | 報名人數 | 目前報名人數 |

這些欄位皆為 optional（`?`），不存在時對應 UI 區塊不渲染。
