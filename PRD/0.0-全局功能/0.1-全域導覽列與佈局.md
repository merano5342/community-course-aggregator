# 全域導覽列與應用程式佈局

---

**頁面位置**：全站共用（所有路由）  
**頁面路由**：所有路由共用 Layout  
**所屬功能模組**：0.0 全局功能  

---

## 需求背景

社大課程整合平台為 SPA（Single Page Application），所有頁面共用一個頂層 Layout 元件。導覽列需在桌面版與行動版提供不同的呈現方式，並即時反映「比較清單數量」與「最愛數量」等全域狀態。

---

## 佈局結構

```
App
├── <Header>          全域頂部導覽列（固定置頂）
│   ├── Logo + 品牌名稱
│   ├── 主要導覽連結（桌面版）
│   ├── 快速數量徽章（最愛、比較）
│   └── 深色模式切換
├── <Outlet>          頁面內容區（React Router）
├── <CompareBar>      比較浮動列（選了 1–3 門課時顯示）
└── <Toaster>         全域 Toast 通知層
```

---

## 頂部導覽列（Header）

### 桌面版（≥ 768px）

| 區塊 | 說明 |
|------|------|
| **Logo 區** | 圖示 + 「社大課程整合平台」文字，點擊導至 `/` |
| **主選單** | 水平連結列：課程列表、比較、我的最愛、時間表、關於 |
| **最愛徽章** | 愛心圖示 + 收藏數量，點擊導至 `/favorites` |
| **比較徽章** | 比較圖示 + 選課數量（0–3），點擊導至 `/compare` |
| **深色模式** | 切換按鈕（sun/moon icon），狀態存 localStorage |

### 行動版（< 768px）

| 區塊 | 說明 |
|------|------|
| **Logo** | 簡短圖示版，置左 |
| **漢堡選單按鈕** | 右側，點擊開啟 Drawer |
| **快速圖示** | 最愛 + 比較的圖示徽章（不含文字），置於漢堡按鈕左側 |

#### 行動 Drawer

點擊漢堡按鈕後，從右側滑入側邊選單：
- 課程列表 `/`
- 跨校比較 `/compare`（含已選數量徽章）
- 我的最愛 `/favorites`（含收藏數量徽章）
- 時間表 `/schedule`
- 關於 `/about`
- 深色模式切換列
- 點擊連結後自動關閉 Drawer

---

## 比較浮動列（CompareBar）

**顯示條件**：比較清單 ≥ 1 門課，且當前路由不是 `/compare`

| 元素 | 說明 |
|------|------|
| **浮動位置** | 頁面底部固定（position: fixed, bottom: 0），全寬 |
| **課程縮覽** | 已選課程名稱（最多 3 個，超過截斷加…） |
| **數量文字** | 「已選 X 門課（最多 3 門）」 |
| **前往比較按鈕** | Primary，點擊導至 `/compare` |
| **清除全部** | Ghost 按鈕，清空比較清單 |

**不顯示情境**：比較清單為空 / 目前在 `/compare` 頁

---

## 全域 Toast 通知

使用 shadcn/ui Toast + Sonner，統一處理以下事件通知：

| 事件 | 訊息範例 | 持續時間 |
|------|---------|---------|
| 加入最愛 | 「已加入最愛：{課程名稱}」 | 2s |
| 移除最愛 | 「已移除：{課程名稱}」 | 2s |
| 加入比較 | 「已加入比較清單」 | 2s |
| 比較清單已滿 | 「比較清單已達 3 門課上限」 | 3s |
| 資料載入失敗 | 「課程資料載入失敗，請稍後再試」 | 5s，含重試連結 |

## 路由結構

| 路由 | 頁面元件 | 說明 |
|------|---------|------|
| `/` | CoursesPage | 課程列表頁（主頁） |
| `/compare` | ComparePage | 跨校比較頁 |
| `/favorites` | FavoritesPage | 我的最愛頁 |
| `/schedule` | SchedulePage | 時間表頁 |
| `/about` | AboutPage | 關於-資料來源頁 |
| `*` | NotFoundPage | 404 頁（含回首頁按鈕） |

GitHub Pages SPA 處理：`public/404.html` 導回 `index.html`，由 React Router 接手路由解析。

---

## 深色模式

- 使用 Tailwind `darkMode: 'class'`，在 `<html>` 加 `dark` class 切換
- 初始狀態：讀取 localStorage `cca_theme`；若無，依 `prefers-color-scheme` 決定
- 切換時同步寫入 localStorage

---

## 響應式斷點

| 斷點 | 寬度 | 行為 |
|------|------|------|
| mobile | < 768px | 漢堡選單；卡片單欄；篩選列改用 Drawer |
| tablet | 768–1024px | 水平選單；卡片雙欄；篩選列側邊 |
| desktop | > 1024px | 水平選單；卡片三欄；篩選列固定側邊欄 |

---

## 效能考量

- Header 使用 `sticky top-0 z-50`，不影響捲動效能
- CompareBar 條件渲染，避免不必要 DOM 掛載
- 深色模式切換用 CSS class，不觸發 re-render
