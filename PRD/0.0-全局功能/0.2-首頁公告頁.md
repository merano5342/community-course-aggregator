# 首頁公告頁

---

**頁面位置**：根路由  
**頁面路由**：`/`  
**所屬功能模組**：0.0 全局功能  
**實作檔案**：`src/pages/HomePage.tsx`

---

## 需求背景

根路由 `/` 作為使用者進入平台的第一個畫面，提供「本學期資訊一覽」而非直接跳入課程清單。內容包含學期時間軸（報名期間、開學日）、各社大發布的公告，以及前往各功能頁面的快速入口。公告篩選偏好儲存於 localStorage，下次進入自動恢復。

---

## 學期 Banner（SemesterBanner）

| 元素 | 說明 |
|------|------|
| **學期名稱** | 如「115學年度第1學期」 |
| **日期範圍** | 開學日～結束日（`YYYY/MM/DD` 格式） |
| **報名倒數** | 今日距報名截止（或報名開始）的天數；已開始顯示「報名中」 |
| **開學倒數** | 今日距開學日的天數；已開學顯示「已開學」 |
| **樣式** | 深色背景卡（`var(--color-text-primary)`），白色文字，底部帶 accent 線 |

倒數計算規則：

| 狀態 | 顯示文字 |
|------|---------|
| 報名尚未開始 | `報名倒數 N 天` |
| 報名中（今日在區間內） | `報名中` |
| 報名已結束 | `報名已截止` |
| 今天開學 | `今天開學！` |
| 開學倒數 | `開學倒數 N 天` |
| 已開學 | `已開學` |

## 快速入口 Grid

2×2 固定格局，各格含圖示 + 文字標籤，點擊直接導覽：

| 格子 | 路由 | 說明 |
|------|------|------|
| 瀏覽課程 | `/courses` | 前往課程列表頁 |
| 我的最愛 | `/favorites` | 前往已收藏課程 |
| 時間表 | `/schedule` | 前往週曆視圖 |
| 跨校比較 | `/compare` | 前往比較頁 |

---

## 社大篩選 Chips

| 元素 | 說明 |
|------|------|
| **「全部」** | 預設選中，顯示所有學校公告 |
| **學校 Chips** | 每所已設定的學校各一個 Chip |
| **多選行為** | 選中特定學校後，「全部」自動取消；可同時選多校 |
| **清除** | 點擊「全部」或所有學校都已取消時，回到「全部」狀態 |
| **持久化** | Zustand persist → `localStorage` key `cca_announcement_filter` |

實作：`src/stores/useAnnouncementFilterStore.ts`（`selectedSchools: string[]`，`toggle(school)`，`clear()`）

---

## 公告卡片（AnnouncementCard）

| 元素 | 說明 |
|------|------|
| **學校 Badge** | 各校固定顏色（和課程列表一致） |
| **類型 Badge** | 報名、時程、停課、場地、一般（對應 `AnnouncementType`） |
| **重要 Badge** | `important === true` 時顯示紅色「重要」Badge |
| **日期** | 右側顯示 `YYYY/MM/DD` |
| **標題** | 粗體，完整顯示 |
| **內容預覽** | 最多 3 行（`-webkit-line-clamp: 3`），超出以「…」截斷 |
| **外部連結** | 若有 `link` 則顯示「查看詳情 →」按鈕，新分頁開啟 |

## 公告資料類型

```typescript
type AnnouncementType = 'registration' | 'schedule' | 'cancel' | 'venue' | 'general';

interface Announcement {
  id: string;
  school: string;      // 學校 id，或 'all'（平台公告）
  type: AnnouncementType;
  title: string;
  content: string;
  date: string;        // YYYY-MM-DD
  important?: boolean;
  link?: string;       // 外部連結（可選）
}
```

類型標籤對照：

| type | 顯示文字 |
|------|---------|
| `registration` | 報名 |
| `schedule` | 時程 |
| `cancel` | 停課 |
| `venue` | 場地 |
| `general` | 公告 |

---

## 公告排序規則

1. `important === true` 的公告優先排在最前
2. 其餘依 `date` 降序（最新的在上方）

## 學期資訊來源

現階段為靜態資料（`src/data/mockAnnouncements.ts`），定義 `CURRENT_SEMESTER: SemesterInfo`：

```typescript
interface SemesterInfo {
  name: string;             // '115學年度第1學期'
  registrationStart: string; // '2026-08-01'
  registrationEnd: string;   // '2026-09-15'
  semesterStart: string;     // '2026-09-07'
  semesterEnd: string;       // '2027-01-16'
}
```

未來爬蟲可從各校首頁擷取並寫入 `data/semester.json`，前端以 TanStack Query 讀取。

## 業務邏輯

- 公告筆數不限制，可捲動查看
