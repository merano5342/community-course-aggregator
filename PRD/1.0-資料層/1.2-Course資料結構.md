## 需求背景

Course schema 是爬蟲輸出與前端消費的共同合約，需在 TypeScript 與 Python 兩側保持一致。必要欄位由爬蟲保證存在；optional 欄位僅在爬蟲成功解析時存在（前端需防禦性渲染）。

## Course Schema

### 必要欄位（Required）

| 欄位名稱 | TypeScript | Python / Pydantic | 說明 |
|---------|-----------|------------------|------|
| `id` | `string` | `str` | `{school}_{hash}` 格式，如 `nangang_8d7ed9df` |
| `school` | `string` | `str` | 學校 id，如 `nangang` |
| `name` | `string` | `str` | 課程完整名稱 |
| `category` | `string` | `str` | 課程類別，如「生活藝能 > 語文類」 |
| `teacher` | `string[]` | `list[str]` | 授課老師列表（可空陣列） |
| `dayOfWeek` | `number` | `int` | 星期，0=日, 1=一, …, 6=六 |
| `timeSlot` | `TimeSlot` | `Literal[...]` | `'morning' \| 'afternoon' \| 'evening'` |
| `startDate` | `string` | `str` | 開課日期，`YYYY-MM-DD` |
| `weeks` | `number` | `int` | 總週數 |
| `fee` | `number` | `int` | 學分費（元），免費為 0 |
| `location` | `string` | `str` | 上課地點 |
| `status` | `CourseStatus` | `Literal[...]` | 見下方狀態定義 |
| `detailUrl` | `string` | `str` | 原校課務系統完整 URL |

### Optional 欄位（v1.1 新增）

| 欄位名稱 | TypeScript | Python / Pydantic | 來源（frog.tw） |
|---------|-----------|------------------|----------------|
| `description` | `string?` | `str \| None` | 課程簡介（理念 + 目標合併） |
| `outline` | `string[]?` | `list[str] \| None` | 逐週大綱，每項格式「第 N–M 週｜主題」 |
| `targetAudience` | `string?` | `str \| None` | 課前資訊 > 適合對象 |
| `teacherBio` | `string?` | `str \| None` | 講師介紹 > 簡介 |
| `notes` | `string?` | `str \| None` | 課前資訊 > 其他備註 |
| `quota` | `number?` | `int \| None` | 招生人數上限 |
| `enrolled` | `number?` | `int \| None` | 目前報名人數 |
| `discount` | `string?` | `str \| None` | 折扣資訊，如 `'80%'`、`'50%'` |
| `isNew` | `boolean?` | `bool \| None` | 是否為新開課程 |
| `isMixed` | `boolean?` | `bool \| None` | 是否為線上+實體混成 |

## 類型枚舉

### CourseStatus

```typescript
type CourseStatus = 'open' | 'full' | 'cancelled' | 'delayed';
```

| 值 | 說明 | frog.tw 判斷依據 |
|----|------|----------------|
| `open` | 開放報名 | 報名人數 < 招生人數，且課程未取消 |
| `full` | 名額已滿 | 報名人數 ≥ 招生人數 |
| `cancelled` | 已取消 | 課程頁標示停課 |
| `delayed` | 延後開課 | 課程頁標示延後 |

### TimeSlot

```typescript
type TimeSlot = 'morning' | 'afternoon' | 'evening';
```

| 值 | 時間區間 | 對應文字 |
|----|---------|---------|
| `morning` | 08:00–12:00 | 上午 |
| `afternoon` | 12:00–17:00 | 下午 |
| `evening` | 17:00–22:00 | 晚上 |

## TypeScript 完整定義

```typescript
// src/types/course.ts

export type CourseStatus = 'open' | 'full' | 'cancelled' | 'delayed';
export type TimeSlot = 'morning' | 'afternoon' | 'evening';

export interface Course {
  // Required
  id: string;
  school: string;
  name: string;
  category: string;
  teacher: string[];
  dayOfWeek: number;
  timeSlot: TimeSlot;
  startDate: string;
  weeks: number;
  fee: number;
  location: string;
  status: CourseStatus;
  detailUrl: string;

  // Optional（v1.1）
  description?: string;
  outline?: string[];
  targetAudience?: string;
  teacherBio?: string;
  notes?: string;
  quota?: number;
  enrolled?: number;
  discount?: string;
  isNew?: boolean;
  isMixed?: boolean;
}
```

## Python Pydantic 定義

```python
# scraper/models.py

from pydantic import BaseModel
from typing import Literal

class Course(BaseModel):
    id: str
    school: str
    name: str
    category: str
    teacher: list[str]
    day_of_week: int          # 序列化為 dayOfWeek（alias）
    time_slot: Literal['morning', 'afternoon', 'evening']
    start_date: str           # YYYY-MM-DD
    weeks: int
    fee: int
    location: str
    status: Literal['open', 'full', 'cancelled', 'delayed']
    detail_url: str

    description: str | None = None
    outline: list[str] | None = None
    target_audience: str | None = None
    teacher_bio: str | None = None
    notes: str | None = None
    quota: int | None = None
    enrolled: int | None = None
    discount: str | None = None
    is_new: bool | None = None
    is_mixed: bool | None = None

    model_config = {
        'populate_by_name': True,
        'alias_generator': lambda s: ''.join(
            w.capitalize() if i else w for i, w in enumerate(s.split('_'))
        ),  # snake_case → camelCase
    }
```

輸出時使用 `model.model_dump(by_alias=True, exclude_none=True)` 確保欄位名為 camelCase 且省略 `null` 值。

## frog.tw HTML → Course 欄位對應

### 列表頁（m_course_list.php）

| Course 欄位 | frog.tw HTML 位置 |
|------------|-----------------|
| `name` | 課程格子 tooltip 或格子內文字 |
| `dayOfWeek` | 格子所在星期欄 |
| `quota` | tooltip「招生人數：N」 |
| `enrolled` | tooltip「報名人數：N」 |
| `location` | tooltip 地點欄 |
| hash（→ `id`、`detailUrl`） | `<a href="m_course_detail.php?u={hash}">` |

### 詳細頁（m_course_detail.php）

| Course 欄位 | frog.tw HTML 位置 |
|------------|-----------------|
| `name` | `<h1>` 或頁面標題 |
| `category` | 麵包屑導覽或標題區 |
| `teacher` | 講師介紹區塊的姓名 |
| `timeSlot` | 開課時間段（上午/下午/晚上） |
| `startDate` | 課程基本資訊 |
| `weeks` | 課程基本資訊 |
| `fee` | 課程費用 > 學分費 |
| `description` | 課程簡介（理念 + 目標段落） |
| `outline` | 課程大綱表格（逐行解析） |
| `targetAudience` | 課前資訊 > 適合對象 |
| `teacherBio` | 講師介紹 > 簡介欄 |
| `notes` | 課前資訊 > 其他 |
| `isNew` | 標題區有「NEW」標記 |
| `isMixed` | 課程屬性有「混成」標記 |

## Announcement Schema

```typescript
// src/types/announcement.ts

export type AnnouncementType = 'registration' | 'schedule' | 'cancel' | 'venue' | 'general';

export interface Announcement {
  id: string;
  school: string;       // 學校 id 或 'all'（平台公告）
  type: AnnouncementType;
  title: string;
  content: string;
  date: string;         // YYYY-MM-DD
  important?: boolean;
  link?: string;
}

export interface SemesterInfo {
  name: string;              // '115學年度第1學期'
  registrationStart: string; // YYYY-MM-DD
  registrationEnd: string;
  semesterStart: string;
  semesterEnd: string;
}
```
